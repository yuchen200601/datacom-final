# 租事大吉 (ZuShiDaJi) - 租屋契約 AI 審閱助手 🛡️

「租事大吉」是一款專為租屋族設計的智慧型合約審閱網頁工具。透過 AI 深度解析租約，協助租屋族在簽約當下揪出法律陷阱，實現居住正義。

---

## 🚀 專案背景
本專案為期末專案，原計畫開發行動應用程式 (App)，考量到簽約現場的即時需求與跨平台部署便利性，最終轉型為**響應式網頁應用程式**。使用者無需安裝任何軟體，開啟瀏覽器即可即時分析租屋合約。

---

## ✨ 功能特色
* **AI 智能風險偵測**：逐條比對台灣現行法規（定型化契約應記載事項、民法），快速識別潛在法律地雷。
* **風險視覺化燈號**：
    * 🔴 **紅色**：明確違法（如：禁報稅、電費超標）。
    * 🟡 **黃色**：合法但不利房客（如：修繕責任完全轉嫁）。
    * 🔵 **藍色**：重要權益提醒。
* **隱私保護機制**：內建 Regex 去識別化功能，自動遮蔽合約中的身分證、電話等個資。
* **自動容錯與切換**：具備備用處理線路，確保在流量高峰期服務依然穩定。

---

## 🛠️ 使用指南


### 1. 網站操作
1. 將合約內容複製並貼入網頁中間的編輯框。
2. 點擊 **「開始線上審閱」** 按鈕。
3. 等待系統進行 OCR 比對與分析，右側面板將列出詳細的風險分析報告。
