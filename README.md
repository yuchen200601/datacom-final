# datacom-final

0. 簡介
原本打算做一個租屋 App，但礙於開發時程與跨平台整合的挑戰，我改成開發成網頁版工具。這樣也比較方便，不用下載和安裝。

1. 系統介紹

1-1 這是什麼？
一個 AI 輔助租屋工具，專門幫你「看懂」合約。你只需要把房東給的條文貼進去，它就能幫你揪出那些隱藏的法律地雷，讓你不再害怕簽下不平等的條約。

1-2 功能亮點
AI 風險偵測： 逐條比對法規，直接告訴你這條合約有沒有問題。
風險燈號提醒：
🔴 紅色（紅燈）： 明顯違法條款（如：禁止報稅、電費亂收）。
🟡 黃色（黃燈）： 合法但很坑人的條款（如：修繕責任全都推給房客）。
🔵 藍色（藍燈）： 重要權益提醒。
自動保護隱私： 內建個資遮蔽機制，身分證、手機號碼貼上去就會自動遮起來，不用擔心資料外洩。
自動容錯： 如果主系統塞車，它會自動幫你切換備用處理線路，確保報告不中斷。

2. 怎麼使用？

2-1 網站操作
貼合約： 把租屋契約內容複製，貼進網頁中間的框框。
審閱： 點擊「開始線上審閱」按鈕。
看結果： 右側面板會立刻跳出分析報告，哪裡有問題一目瞭然。

2-2 變數與環境設定（開發者注意）
為了讓你的網頁能正確連結 AI 模型，請記得修改程式碼中的設定：
Cloudflare Worker 網址： 找到 HTML 檔案裡的 WORKER_URL，把你的 Cloudflare Worker 專屬網址填進去。

JavaScript
// 請改成你的真實 Worker 網址
const WORKER_URL = "https://你的_worker_網址.workers.dev";
API 金鑰： 如果使用到 GitHub API，請記得在後端環境設定（.env）填入你的 GITHUB_TOKEN，確保 AI 服務正常連線。
